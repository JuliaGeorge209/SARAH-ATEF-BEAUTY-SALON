# 💅 SARAH-ATEF-BEAUTY-SALON

موقع إلكتروني متكامل لإدارة وعرض خدمات صالون التجميل الخاص بـ **سارة عاطف**، يتيح للمستخدمين تصفح الخدمات المتاحة، وللإدارة التحكم في المحتوى من خلال لوحة تحكم مخصصة.

## 🚀 التقنيات المستخدمة (Technologies Used)

* **الواجهة الأمامية (Frontend):** HTML5, CSS3, JavaScript
* **الخادم (Backend):** TypeScript, Express.js
* **البيئة والنشر:** Node.js, Vercel (`vercel.json`), GitHub Actions (`CI/CD`)

---

## 📂 هيكلة المشروع (Project Structure)

* `index.html` - الصفحة الرئيسية للمستخدمين لتصفح الخدمات (مثل أظافر جل، رموش فوليوم، نقش يدوي).
* `admin.html` - لوحة التحكم الخاصة بمدير الصالون لتحديث البيانات.
* `server.ts` - ملف الخادم المبني بـ TypeScript لإدارة الطلبات (Requests).
* `styles.css` & `script.js` - ملفات التنسيق والتفاعلات الخاصة بالواجهة.
